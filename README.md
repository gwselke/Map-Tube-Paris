A Map::Tube implementation for the Paris metro, RER, Transiliens, tram, and funicular lines.

This module allows to find the shortest route between any two given local or
regional transport stations in Paris.

All interesting methods are provided by the role L<Map::Tube>.

To build this module, use the classical steps:

* perl Makefile.PL
* make
* make test
* make install

(If you are using Srawberry Perl under Windows, you may want to replace "make"
with "gmake".)
